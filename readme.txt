Contact Form 7 Repeater adds extra functionality to Contact Form 7 which will allow you to repeat all types of fields and validate each one of them easily.

== Description ==

Contact Form 7 Repeater will allow you to repeat all kinds of fields from text, files, checkboxes, radio buttons, textarea etc..., you can also validate each one of them by using CF7 validation, This plugin uses CF7 Plugin to Parse, Submit the repeater and Validate the fields.

= Requirements =

* Requires contact form 7 v4.4 or higher
* Repeat any field or set of fields you want using jQuery and submit them with validation of each field.

= Features =

* Repeat all types of fields
* Limit minimum and maximum fields to repeat
* Control the fields limit, show and hide
* Validation per each fields
* Doesn't require any special setup
* Shortcodes are handled with Contact Form 7, The plugin is simple and light.
* Lite Version allows 1 repeater only

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/contact-form-7-repeater` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. The plugin is functional with one repeater, a license is required to unlock more than one repeater.
